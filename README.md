# Cell Blocks
cellular automata sandbox
conway game of life rules

## Setup Instructions
clone repo
run parcel commands

## Usage

buttons:
-start/pause
-reset
-randomize

conway rules:
-neighbors to live
-neighbors to die

clicking canvas:
-click or drag to toggle cells

patterns:
-drag and drop
